## 💻 활동 내용
- **✏️ 개인 미션**
매일 1문제 이상 풀고 밤 11시 50분까지 카톡방에 인증샷
주말의 경우 통합 1문제 이상 (토 00:00 ~ 일 23:50)
- **🚀 공통 미션**
스터디원이 돌아가면서 문제를 선정하며, 해당 문제를 다음 스터디일까지 풀이
    >혜선 → 선민 → 예린 → 성현 → 이루
- 매주 **화, 금** 정규 일정 후에 대면으로 코드 리뷰 & 토론
